# 实验一：操作系统的启动

!!! tip "就像宇宙的大爆炸，操作系统的启动是一切的起点。<br/>在这个微妙的瞬间，一切从无到有，从混沌到秩序。<br/>&nbsp;<span style="font-weight: bold; float: right">by Copilot</span>"

## 实验目的

1. 了解页表的作用、ELF 文件格式、操作系统在 x86 架构的基本启动过程。
2. 尝试使用 UEFI 加载并跳转到内核执行内核代码。
3. 实现基于 uart16550 的串口驱动，使用宏启用输出能力、并启用日志系统。
4. 学习并尝试使用调试器对内核进行调试。

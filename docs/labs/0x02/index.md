## 实验目的

1. 了解中断的作用、中断的分类、中断的处理过程。
2. 实现基于 APIC 的中断处理程序，编辑 IDT 实现时钟中断，实现一个跑马灯。
3. 注册内核堆分配器。（不实现内存分配算法，使用现有代码赋予内核堆分配能力）
4. 实现串口驱动的输入能力，尝试进行基础的 IO 操作和交互。

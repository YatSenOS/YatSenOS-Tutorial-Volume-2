## 实验目的

1. 了解文件系统的概念、作用。
2. 实现块设备、磁盘、分区、文件系统的抽象。
3. 了解 ATA 硬盘的工作原理、实现基本的读写驱动。
4. 实现 FAT16 文件系统的读取和只读文件访问。
